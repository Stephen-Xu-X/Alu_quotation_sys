# Alu_quotation_sys

# 💡 铝型材报价系统 · 产品深度优化建议书
**版本号**：v1.0
**状态**：已归档
**适用阶段**：MVP 后期迭代 (v1.5 - v2.0)

---

## 一、 核心目标
从“基础计算工具”向“商业化销售辅助神器”跨越，重点提升**用户粘性**、**操作效率**与**传播价值**。

---

## 二、 优化维度与详细方案

### 1️⃣ 用户体验 (UX) 与交互细节
> **目标**：降低重复劳动，提供物理级的操作爽感。

* **P0 | 数据智能记忆 (Local Storage)**
    * **痛点**：销售员每日基价、税率固定，重复输入繁琐。
    * **方案**：
        1.  监听 `blur` 或 `input` 事件，将基价、税率、利润系数自动存入 `wx.setStorageSync`。
        2.  `onLoad` 时自动读取缓存回填。
    * **预期收益**：单次报价操作时间缩短 50%。

* **P0 | 触感反馈系统 (Haptic Feedback)**
    * **方案**：在关键节点（计算结果变化、点击复制、卡片翻转、清空数据）调用 `wx.vibrateShort({ type: 'light' })`。
    * **预期收益**：赋予小程序“实体工具”般的精致质感。

* **P1 | 防误触与“后悔药” (Undo Mechanism)**
    * **方案**：点击“清空”后，显示顶部 Snackbar 提示“数据已重置”，并提供 [撤销] 按钮，点击可恢复上一刻数据。

### 2️⃣ 视觉表现 (UI) 与工程化
> **目标**：强化视觉层级，利用微信生态优势。

* **P1 | 报价海报生成 (Canvas/Painter)**
    * **场景**：销售需将正式报价单发送至微信群或朋友圈。
    * **方案**：引入 Canvas 绘制模块，生成包含“公司Logo + 实时报价 + 详细参数 + 二维码”的精美图片，支持一键保存相册。
    * **预期收益**：极大提升品牌曝光度与专业度。

* **P2 | 结果展示视觉分层**
    * **方案**：
        1.  **货币符号解耦**：`¥` 字号缩小至 `40rpx`，数字保持 `80rpx`，对齐基线。
        2.  **动态色温**：利润系数 > 1.3 时金额微泛暖色（红），< 1.05 时微泛冷色（绿/蓝），暗示利润空间。

* **P2 | 交互微动效**
    * **方案**：工程师模式计算米重应用回正面时，正面“米重”输入框执行一次 `Highlight Flash`（高亮闪烁），提供明确的视觉反馈。

### 3️⃣ 业务逻辑扩展
> **目标**：增强工具的权威性与依赖性。

* **P1 | 历史报价记录 (History Log)**
    * **方案**：在“快捷工具”中增加入口，本地存储最近 50 条报价记录（含时间戳、客户名、最终价），支持点击回溯复用。

* **P2 | 实时铝价接入 (Real-time API)**
    * **方案**：接入“南海灵通”或“长江有色”API，在基价输入框下方显示当日实时均价与涨跌幅。
    * **文案示例**：`今日南海灵通：20,480 (↑20)`
